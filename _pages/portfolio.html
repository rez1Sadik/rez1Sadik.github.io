Aspect-Based Sentiment Analysis of Amazon Reviews

Developed a high-accuracy NLP pipeline to extract aspect-based sentiments from Amazon product reviews. This project utilized a hybrid feature engineering approach, combining TF-IDF with n-grams to deeply analyze customer feedback on electronics. Evaluated multiple machine learning models, ultimately achieving a 91% accuracy rate using a Support Vector Machine (SVM) algorithm, while effectively managing dataset imbalance (validated by a 0.77 MCC score).
